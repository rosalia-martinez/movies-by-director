# 🎥 movies-by-director (Español)

## 📝 Descripción

Este repositorio contiene el proyecto colaborativo **movies-by-director**, una aplicación web que permite explorar películas organizadas por director. Es un proyecto de clase enfocado en practicar el trabajo en equipo con Git y GitHub, así como el desarrollo frontend.

## 🎯 Objetivo del Proyecto

Construir una interfaz web que liste directores y, al seleccionar uno, muestre sus películas más representativas. El objetivo es aprender a:

- Colaborar eficazmente usando ramas, pull requests y resolución de conflictos.
- Maquetar una interfaz clara y responsive.
- Organizar el código de forma modular y reutilizable.

## ⚙️ Tecnologías

| Capa                 | Tecnologías                 |
|----------------------|-----------------------------|
| Estructura           | HTML5                       |
| Estilos              | CSS3                        |
| Control de versiones | Git y GitHub                |
| Entorno local        | Visual Studio Code          |

## 🗂️ Arquitectura del Proyecto

Hemos organizado el proyecto en una estructura clara para evitar conflictos y facilitar la escalabilidad:
movies-by-director/
├── index.html # Página principal
├── pages/ # Páginas por director
│ ├── tim-burton.html
│ ├── hayao-miyazaki.html
│ └── ...
├── assets/
│ ├── css/
│ │ └── style.css

└── README.md

# 🎥 movies-by-director (English)

## 📝 Description

This repository contains the collaborative project **movies-by-director**, a web application that allows users to explore movies organized by director. It is a class project focused on practicing teamwork with Git and GitHub, as well as frontend development.

## 🎯 Project Goal

Build a web interface that lists directors and, upon selection, displays their most representative films. The goal is to learn how to:

- Collaborate effectively using branches, pull requests, and conflict resolution.
- Create a clean, responsive interface.
- Organize code in a modular and reusable way.

## ⚙️ Technologies
